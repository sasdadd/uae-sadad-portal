# بوابة سداد الإمارات
## UAE Sadad Portal - Electronic Payment Gateway

منصة آمنة وموثوقة لسداد الفواتير الحكومية والخاصة في دولة الإمارات العربية المتحدة.

### الميزات الرئيسية
- ✅ سداد آمن وموثوق مع أحدث معايير التشفير
- ✅ دعم جميع البنوك الإماراتية الرئيسية
- ✅ واجهة سهلة الاستخدام باللغة العربية
- ✅ خدمات متعددة (فواتير حكومية، خدمات خاصة)
- ✅ متاح 24/7 من أي مكان

### التقنيات المستخدمة
- **Frontend:** React 19 + TypeScript + Vite
- **Styling:** Tailwind CSS 4
- **UI Components:** shadcn/ui
- **Icons:** Lucide React
- **Routing:** Wouter

### البدء السريع

```bash
# تثبيت المتطلبات
pnpm install

# تشغيل خادم التطوير
pnpm dev

# بناء للإنتاج
pnpm build

# معاينة الإنتاج
pnpm preview
```

### هيكل المشروع

```
client/
├── public/          # ملفات ثابتة
├── src/
│   ├── pages/       # صفحات التطبيق
│   ├── components/  # مكونات قابلة لإعادة الاستخدام
│   ├── contexts/    # React Contexts
│   ├── hooks/       # Custom Hooks
│   ├── lib/         # دوال مساعدة
│   ├── App.tsx      # المكون الرئيسي
│   ├── main.tsx     # نقطة الدخول
│   └── index.css    # الأنماط العامة
server/              # خادم Express (اختياري)
```

### الصفحات الرئيسية
- **الصفحة الرئيسية:** عرض الخدمات والميزات
- **سداد آمن:** معلومات الأمان والحماية
- **البنوك المدعومة:** قائمة البنوك الإماراتية
- **الدعم:** مركز المساعدة والأسئلة الشائعة
